# SignMap Microservices Starter

## Cấu trúc
- Contribution Service: Quản lý góp ý biển báo  
- Payment Service: Quản lý giao dịch thanh toán  
- MongoDB: Lưu dữ liệu chung  

## Chạy bằng Docker
```
docker compose up --build
```

- Contribution API: http://localhost:8081/api/contributions  
- Payment API: http://localhost:8082/api/payments
